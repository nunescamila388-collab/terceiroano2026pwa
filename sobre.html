Sobre o Projeto

O que é um PWA?

PWA significa Progressive Web App (Aplicação Web Progressiva). É um tipo de site que pode funcionar como um aplicativo no celular.

Vantagens

  • Funciona offline
  • Pode ser instalado no celular
  • É rápido e leve
  • Não precisa baixar na loja

Sobre este projeto

Este projeto foi desenvolvido na aula do Senac com o objetivo de aprender a criar aplicações web modernas utilizando HTML, CSS e conceitos de PWA.
